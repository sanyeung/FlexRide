The two most important source files in the tsharesim/src folder are:
1. simulator.cpp
2. treeSlackTaxiPath.cpp

To compile the simulator, navigate to the tsharesim/src directory and run the command "make"

To run the simulator, navigate to the tsharesim/src dircectory and run the command "./simulator 4"

For information on generating valid vertices/edges, view the readme in the data_utilities directory

For information on generating valid traffic updates, view the readme in the traffic_generator directory
